1. 不要動到 source code
2. 創建 uv 環境 更改解釋器路徑 uv 的 python 
3. 使用 vanna 的框架 pip install "vanna[chormadb]" "vanna[ollama]" "vanna[openai]"





-  進到我的資料夾

    cd "專案名稱"

- 初始化專案(如果還沒有 pyproject.toml)

    uv init

- 建立虛擬環境

    uv venv

- 啟用虛擬環境

    (bash) source .venv/Scripts/activate

    (powershell) .venv/Scripts/activate 

- 安裝套件

    uv add pandas

- 停用虛擬環境

    deactivate
